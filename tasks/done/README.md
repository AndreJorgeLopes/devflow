# Done Tasks

This folder contains completed task files moved from P0–P4 priority folders after implementation.

Tasks are moved here (not deleted) to maintain a record of what was built, why, and what files were changed.

## Convention

Files follow the same frontmatter format as active tasks, with `status: done` and `completed_date` added.

Original task files in P0-P4 folders are left in place and their `status` field should be updated to `done` when possible.
